# ⚙️ TPC'AI Gerador

Gerador de animações em HTML + CSS usando inteligência artificial.

👉 Acesse: https://seu-link-aqui.com

---

## ✨ O que faz

- Gera animações a partir de texto  
- Exibe preview em tempo real  
- Permite copiar o código gerado  

---

## 🧪 Exemplo

Digite:
"Bola quicando"

E receba uma animação pronta automaticamente.

---

## 🛠️ Tecnologias

- HTML, CSS, JavaScript  
- Node.js + Express  
- API Groq (LLaMA 3)
